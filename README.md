# cyberincident-news-feeder
CyberIncident-news-Feeder is a tool designed to scrape and collect real-time feeds of cyber incidents specifically related to the Indian cyberspace.This tool leverages web scraping techniques using Selenium to gather, parse, and store cyber incident data in a structured format and display it to user.

User can identify potential vulnerabilities, have quick access to relevant cybersecurity news and can take further actions to it.
Real-time Web Scraping: Utilizes Selenium to scrape news and incident data from multiple sources.
User can also scrape the real data and view it, this also distinguishes the different source of user which can be benificial for the user.
This is very helpful in the current scenarios where the internet security is much needed, most of the cases happen because people are unaware, this project helps in knowing the current CyberIncident and trends.
